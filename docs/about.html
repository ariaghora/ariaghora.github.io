About this website

This website is my personal website, where I write about my projects, my thoughts, and sometimes my life. Though I wish this website can be useful for others, I mostly write to express myself as the priority.

Errata policy

I am trying my best to make this website as error-free as possible, and filling it with timeless content. If you find any errors (typographical, code, math equation, anything), please let me know by one of the following ways:

  • Open an issue on GitHub
  • Send me an email at hello@ghora.net
  • Send me a message on Twitter

It doesn't matter whether the typo is in very old articles or not. I appreciate any help in making this website better.
